node & npm

בשביל שהמחשב יוכל להריץ javascript צריך להתקין nodejs ,
ביחד עם node מותקן גם תוכנה שמאפשרת להתקין בקלות ספריות javascript שנקראת npm.

command line

כל הניהול היצירה והבניה של אנגולר נעשה על ידי שרת הפקודה,
בכל בקום באתר נתייחס לשורת הפקודה בכינוי cli ,
אם אתם לא יודעים כיצד להשתמש בשורת הפקודה קיראו את המדריך הבא יש שם גם הסבר קצר על command line.

angular cli

angular cli זה הספריה של אנגולר שמאפשרת לבנות פרוייקטים של אנגולר,
טכנית אפשר ליצור פרוייקט של אנגולר שיעבוד גם בלי angular cli ,אבל זה דומה לתכנת מערכת הפעלה דרך פינקס רשימות, אפשרי אבל לא כל כך יעיל,
angular cli התקנה: הקלידו ב cli npm install -g @angular/cli

typeScript

typeScript זה הרחבה ל javascript, לקבצים של typeScript יש סיומת .ts,
typeScript הופך את את javascript משפה פונקצוניאלית למונחת עצמים,
typeScript לא מוסיפה פונקצוניאלית לשפה אלא רק עושה סדר בקוד על ידי הוספה של קלאסים interface ירושות וכמובן types כמו string או number ולא רק var,
typeScript לא יכול לרוץ על הדפדפן כמן javascript, אלא חייב לעבור תהליך המרה ל javascript, כך שלמעשה הקוד שיעבוד בשרת בפועל זה לא מה שכתבנו אלא קובץ מצוקמק של javascript,
גם אם משתמשים בסביבה של typeScript אין שום חובה להשתמש בשפה, למשל אם לא תקבעו type למשתנה לא תקבלו על זה שגיאה,
אנגולר כוללת בתוכה typeScript, והארכיטקטורה של אנגולר בנויה על typeScript, אין חובה להשתמש ב typeScript, אבל איך נאמר אין גם חובה להשתמש באנגולר.
בהמשך יש מדריך על המידע המינימלי שצריך בשביל להשתמש באנגולר,
