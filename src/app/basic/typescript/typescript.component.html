import & export

בשביל שיהיה אפשר להעתיק אובייקט כלשהו מקובץ לקובץ אחר צריך להשתמש ב export ו import,
לפני האובייקט שרוצים ליצא צריך להוסיף export לדוגמא export counter = 1,
ובקובץ המייבא מייבאים בעזרת import, צורת הכתיבה היא קודם import אחרי זה סוגריים מסולסלות עם שם האובייקט המיובא ואחרי זה המילה from ואחרי זה string הנתיב לקובץ המייצא,

observable

observable זה אובייקט שמחזיק משתנה שאפשר להגדיר שאם הערך שבמשתנה השתנה תבוצע פונקציה, במילים האחרות observable נותן אופציה להאזין למשתנה,
observable מגיע מספריה בשם ReactiveX ובקיצור rxjs,
באנגולר יש הרבה שימוש ב observable לדוגמא בajax,
דוגמא:

הסבר:

יצירה של observable ( ההסבר יהיה בהנחה שה observable מוגדר בתוך קלאס):
1) מייבאים את אובייקט observable כך import {{'{'}} Subject } from 'rxjs/Subject' (בהנחה שאתם בתוך פרוייקט של אנגולר הספריה ממוקמת בכתובת הזאת) ,
2) יוצרים משתנה רגיל נקרא לו myVar ,
3) יוצרים אובייקט observable כמו זה myObservable = new Subject() ,
4) בתוך מתודה מכניסים למשתנה הרגיל את האובייקט שהוא observable עם המתודה asObservable()., זה נראה כך: {{' method() { this.myVar = myObservable.asObservable() } '}} ,

גישה ל observable:
מוסיפים ל observable את המתודה subscribe זה נראה כך: {{' myObservable.subscribe(val)=> { consoel.log(val)}' }} ,
מהרגע שתופעל המתודה שמכניסה את ה observable למשתנה בכל פעם שהערך של ה observable ישתנה המתודה subscribe תופעל,
