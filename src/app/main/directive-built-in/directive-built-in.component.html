סקירה

directive הם attribute שנוצרו על ידי אנגולר,
אפשר ליצור directive חדשים וישנם directive חדשים שבאים עם אנגולר, בפרק זה נתמקד באלה שכבר קיימים,

ngFor*

מה זה עושה:

ngFor* מציג מערכים, הוא מכפיל את התג שבו הוא נמצא כמספר התאים במערך כשבכל תג שנוצר ניתן לשים בו ערך של תא במערך, למעשה זה לולאה בhtml,
לדוגמא בקוד: {{ ngForExample1 }}, בתצוגה: {{ ngForExample2 }},

אפשרויות נוספות

ngFor - index

מה זה עושה:

יש אפשרות להוסיף index ללולאה זה נראהבתצוגה כך: {{ indexExample }},

ngIf*

מה זה עושה:

מוחק את התג שבו הוא נמצא אם מקבל ערך בוליאני false או null או undefined,
כך זה נראה בתצוגה: {{ ngIfExample }},

אפשרויות נוספות

ngIf - else

מה זה עושה:

ניתן להגדיר תג html בשם ng-template שיופיע אם הngIf מקבל ערך false
כך זה נראה בתצוגה: {{ ngIfElseExample }},

ngIf - then

ניתן להגדיר תג html בשם ng-templateשיופיע אם הngIf מקבל ערך true
כך זה נראה בתצוגה: {{ ngIfThenExample }},

ngClass

מה זה עושה:

ngClass מפעיל css class שהוגדר מראש במידה שקיבל true,

איך כותבים:

הdirective ngClass מקבל אובייקט שהkey שלו זה שם הקלאס והvalue שלו זה ערך בוליאני של true של false,
כך זה נראה בתצוגה: {{ ngClassExample }},

ngStyle

מה זה עושה:

ngStyle מפעיל style חדש,

איך כותבים:

הdirective ngStyle מקבל אובייקט שכתוב כמו אובייקט של קלאס css,
כך זה נראה בתצוגה: {{ ngStyleExample }},

ngSwitch
