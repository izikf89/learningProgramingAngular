ניתן להגדיר בcomponent ןdirective attribute נוסף שיוכל לקבל ערך ולהעביר אותו לקוד

יש שני דרכים איך להגדיר:

1) input decorated

ניתן להגדיר על ידי מתודה decorated בתוך הקלאס
דבר ראשון יש לייבא את המתודה @Input כך: {{ importInput }}
ההגדרה של הinput נראית כך:@Input('attributeName') propertyName;

ניתן גם להגדיר בצורה יותר קצרה כשהשם של הattribute והproperty הוא אותו אחד כך: @Input() attributeName;

2) input property

ניתן להגדיר את הattribute הנוסף בproperty input במתודה decorated של הקלאס כך: inputProperty

ניתן גם להגדיר בצורה יותר קצרה כשהשם של הattribute והproperty הוא אותו אחד כך: inputs: [ 'newAttribute'],
