הsyntex של event באנגולר,

בחלק מהדוגמאות נעשה שימוש במתודה ngOnInit, המתודה הזאת מובנית באנגולר ומוגדרת שהיא מופעלת באופן אוטומטי כשהקלאס נוצר,

האזנה לevent

יש שלושה דרכים להאזין:

1) בhtml

כמו שמגדירים בhtml רגיל רק שבלי האתיות "on" בתחילת השם של הevent ומוקף בבסוגריים עגולות, (binding מהתצוגה לקוד)
לדוגמא בhtml רגיל event מסוג click נכתב כך: onClick='' ובאנגולר הוא נכתב כך (click)='',

2) HostListener

זה מתודה decorated שמיוחדת להאזנה והיא מוגדרת בתוך הקלאס כך: {{ hostListener }},

הפרמטר $event הוא אופצוינלי, אפשר גם להגדיר בלעדיו כמו בדוגמא הבאה: {{ hostListenerShort }}

3) host

אפשר להגיר את המאזין במתודה decorated בproperty host: {{ host1 }},
ואז להגדיר בקלאס את המתודה שהוגרה בhost עם פרמטר אחד, לדוגמא: {{ host2 }},

enter

enter זה event מיוחד לאנגולר,
הוא קיים רק בתג input,
הוא מופעל כשהפוקוס על הinput ולוחצים על enter, שם הevent הוא keyup.enter,

יצירת event

יש שני דרכים ליצור event:

1) decorated output:

דבר ראשון צריך לייבא את המתודה decorated ואת האובייקט שיוצרים את הevent כמו בדוגמא: {{ importOutputDecorated }},
מגדירים event בקלאס כמו בדוגמא: @Output() eventName = new EventEmitter();
וכך משחררים את הevent: {{ releaseEvent }},

2) property output:

דבר ראשון צריך לייבא את האובייקט שיוצר את הevent כמו בדוגמא: {{ importOutputProperty }},
מגדירים event בproperty outputs של המתודה decorated של הקלאס כמו בדוגמא: outputs: ['eventName'],,
וכך משחררים את הevent: {{ releaseEvent }},
