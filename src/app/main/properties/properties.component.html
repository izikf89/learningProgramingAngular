מה זה עושה:

באנגולר אפשר להגדיר כל property של תג html כמו שמגדירים כל attribute

אם לא ממש הבנתם את המשפט הקודם ולא ממש ברור לכם מה ההבדל בין attribute וproperty או מה זה "property של תג html" דלגו להמשך העמוד לקטע "מה זה property",

איך כותבים:

מתייחסים אל הproperties כמו אל attribute רק שמוסיפים להם סוגריים מרובעות
לדוגמא {{ propertiesExample }},

מה זה property

property זה מה שנמצא בתוך אובייקט לדוגמא {{ objectExample }},
בתוך כל תג html קובעים את ההגדרות של התג כמו לדוגמא id זה נקרא attribute וזה חלק מהhtml,
לjavascript יש תכונה שנקראת DOM שיוצרת נגד כל תג html אובייקט שמכיל את הattribute של התג ואת התגי html שהוא מכיל, ומאפיינים נוספים מעבר לattribute כמו hidden,
כשמשנים תג html עם javascript לא משנים את html אלא את האובייקט שיצר הDOM, הDOM מזהה את השינוי ומעדכן את התג בהתאם,
כשאמרנו properties של תג html הכוונה לproperties של האובייקט html שיצר הDOM,
