בכל ניווט של אנגולר הראוטר משחרר כמה events במהלך הניווט שמעדכנים מה הסטטוס של הניווט,

רשימת האירועים:
  • NavigationStart - כשהניווט מתחיל,
  • RoutesRecognized - כשהראוטר מזהה את הכתובת,
  • RouteConfigLoadStart - לפני שמתחיל לטעון lazy loading,
  • RouteConfigLoadEnd - אחרי טעינה של lazy loading,
  • NavigationEnd - כשהניווט מסתיים בהצלחה,
  • NavigationCancel - כשפונקציה מסוג guard חסמה את הניווט,
  • NavigationError - כשהניווט נכשל,

1) דבר ראשון מייבאים את הservice Router וקלאס של הevent שאנחנו מעוניינים, לדוגמא: {{ import }},
את הservice Router רושמים במתודה constructor, דוגמא: {{ registers }},
2) הservice Router מחזיק property בשם event שהוא אובייקט מסוג observable והוא מתעדכן באירועים של הראוטר, יש להירשם אליו באמצעות המתודה subscribe ולהשוות בין האירועים לאירוע המבוקש,
