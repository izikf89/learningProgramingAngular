בדרך כלל יוצרים אלמנטים בcomponnet ואפשר להשתמש בהם רק בתוך הפרוייקט של אנגולר ולא מעבר לזה, החל מאנגולר 6 ניתן להפוך components לאלמנטים אמיתיים של html כך שיהיה ניתן להשתמש בהם גם מחוץ לאנגולר, הסוג הזה של האלמנטים מכונה web components והרעיון בהם הוא שאפשר יהיה לשלב קוד בין שני ספריות,

דבר ראשון יש להוסיף את הספריה @angular/elements כמו הקוד הבא: ng add @angular/elements
כעת יש לייבא את הספריה כמו בדוגמא: {{ import }},
וצריך לייבא גם הservice injector כמו בדוגמא: {{ importInjector }},
וצריך לייבא גם הservice createCustomElement כמו בדוגמא: {{ importCreateCustomElement }},
רשום את הservice injector במתודה constructor כמו בדוגמא: {{ constructorExample }},
כעת ניצור property שמכיל את הcomponent שאנחנו מעוניינים לעשות אותו html, עושים זאת על ידי שימוש במתודה createCustomElement, מכניסים למתודה שני פרמטרים הראשון זה הcomponent שהופכים לאלמנט, והפרמטר השני זה הinjector, כמו בדוגמא: {{ createCustomElement }},
כעת נעשה מהproperty הנ"ל אלמנט html על ידי המתודה customElements.define(), למתודה נכניס שני פרמטרים, הראשון זה string שהוא יהיה הselctor של האלמנט, והפרמטר השני זה הproperty הנ"ל שמחזיק את הcomponent, כמו בדוגמא: {{ customElements }},
