מאפשר לשלוט במיקום של child שיתווספו על ידי component שלישי

איפה שנמקם את התג ng-content שם יתווסף הchild

ניתן למקם מספר child במקום אחד כמו בדוגמא:

multi child

ניתן למקם סוגי child שונים במיקום שונה בcomponent
הוסיפו לתג ng-content את הattribute select וניתן לו ערך את התג של הchild או id או קלאס שקיים בchild, דוגמא: {{multiChild}}
